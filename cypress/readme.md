# Steps to run the test

# Install Node.js

    a- In cmd run NPM install

# Install Cypress

    a- npm install cypress

# Run Cypress UI

    a- npx cypess open
    b- select E2E Test
    c- select Test case that you want to run

    ![Alt text](image-1.png)

# Run Cypress from console

    a- after firts running in cmd run npx cypress run
